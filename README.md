# dataset-cleaning
In this work, the dataset netfix tv shows and movies was taken and it contains 7000 data and 12 columns.
The missing values were found in the columns like director,cast,country,date_added and rating.
They were filled using some of the forms such as unavailable or unknown or mode values.
The next step was to remove duplicates.Duplicates were also got removed.
Column names cleaned and data types were also verified during the data cleaning process.
The final data set was given in the name cleaned_netflix_dataset.csv.
